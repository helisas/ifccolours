# ifccolours
Tool for assigning IfcSurfaceStyle inside IFC files. Apply colours to elements by property from a specific Pset, by category (IFC class) or by type.

Instruction:

1. Open the terminal window
2. You need to have Python installed on your computer (version from 3.9 to 3.14).
3. You need to have ifcopenshell library installed on your computer.
4. In the terminal, type: cd [path to the folder where you have ifccolours.py file]. For example: cd C:\Users\magda\ifccolours
5. Type python ifccolours.py
6. The program will ask you for the IFC file path. Paste it without any quotation marks etc., just the path, for example: C:\Users\magda\Desktop\ifc\example_file.ifc
7. Continue doing what it tells you
8. At the end, the coloured file will be saved in the same folder as the original one.
